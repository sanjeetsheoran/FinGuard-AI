# FinGuard-AI
An Agentic AI-Powered Financial Intelligence Platform for Real-Time Fraud Detection using Hybrid Deep Learning, Graph Neural Networks, Explainable AI, and Retrieval-Augmented Generation.


## Features

- Real-time Fraud Detection
- AI Risk Scoring
- Explainable AI (SHAP & LIME)
- Agentic AI Investigation
- Financial Knowledge Assistant
- Interactive Dashboard
- Report Generation
